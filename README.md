# 🎮 Игра "Коридор" (Quoridor)

Браузерная версия настольной игры "Коридор" на JavaScript (HTML5 Canvas).

## 📋 Правила игры
- Поле 9×9 клеток
- Игроки стартуют на противоположных сторонах
- Цель: дойти до противоположной стороны первым
- У каждого игрока по 10 стен
- Стены ставятся на границы между клетками (перекрывают 2 клетки)
- Нельзя полностью заблокировать путь соперника

## 🕹️ Управление
- **ПРОБЕЛ** — переключение режима (ходьба / установка стен)
- **Клик мыши** — выполнить действие
- **Наведение мыши** — подсветка доступных ходов и стен
- **ПКМ / Escape** — отмена

## 🤖 Режимы игры
- 2 игрока (PvP)
- ИИ Лёгкий
- ИИ Средний
- ИИ Сложный

## 💾 Сохранение
Результаты игр сохраняются в localStorage браузера.

## 🚀 Запуск
1. Скачайте файл `index.html`
2. Откройте его в любом современном браузере

## 🛠️ Технологии
- HTML5 Canvas
- JavaScript (ES6+)
- LocalStorage API
