# 🎮 Игра "Коридор" (Quoridor)

Браузерная версия настольной игры "Коридор" на чистом JavaScript (HTML5 Canvas).

## 📋 Правила игры
- Поле 9×9 клеток
- Игроки стартуют на противоположных сторонах
- Цель: дойти до противоположной стороны первым
- У каждого игрока по 10 стен
- Стены ставятся на границы между клетками (перекрывают 2 клетки)
- Можно перепрыгивать через соперника
- Нельзя полностью заблокировать путь соперника

## 🕹️ Управление
- **ПРОБЕЛ** — переключение режима (ходьба / установка стен)
- **Клик мыши** — выполнить действие (походить или поставить стену)
- **Наведение мыши** — подсветка доступных ходов и вариантов стен
- **ПКМ / Escape** — отмена

## 🤖 Режимы игры
- **2 игрока (PvP)** — игра вдвоём на одном устройстве
- **ИИ** — игра против компьютера (алгоритм Минимакс с альфа-бета отсечением, глубина 6)

## 📜 История матчей
- Автоматическое сохранение результатов после каждой игры
- Отображение последних 10 матчей (победитель, режим, количество ходов)
- Кнопка очистки истории
- Данные хранятся в localStorage браузера

## 🛠️ Технологии
- **HTML5 Canvas** — отрисовка игрового поля, игроков, стен, подсветки
- **JavaScript (ES6+)** — вся игровая логика и ИИ
- **LocalStorage API** — база данных для хранения истории матчей
- **Алгоритм Минимакс** — искусственный интеллект с альфа-бета отсечением
- **BFS (поиск в ширину)** — поиск кратчайшего пути для оценки позиции

## 🚀 Запуск
1. Скачайте файл `corridor.html`
2. Откройте его в любом современном браузере
3. Игра готова к запуску!

## 📦 Особенности
- Полностью автономный файл (без сервера, без зависимостей)
- Адаптивный интерфейс
- Умный ИИ, который анализирует позицию на 6 ходов вперёд
- Подсветка доступных действий при наведении
